TID

Identity, presence, and protection tooling for systems that need to be seen correctly.

TondID is a local-first platform for configuring and validating transmitted identity behavior on protected moving assets. It is designed for clean operator control, persistent hardware configuration, and repeatable field testing.

Contact Ops

Local-first control

Browser-based configuration over existing USB Type-C, with persistent profile storage on the device.

Hardware-backed testing

ESP32-class firmware with configurable identity, movement, swarm composition, and runtime behavior.

Built for protected systems

Suitable for testing around drones, rail, aviation, heavy equipment, and other systems that depend on trustworthy visible identity.
